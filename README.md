# angela-wolber.de

Website von Angela Wolber – Sprecherin für Hörbücher & Voice Over.

Statische Website (HTML/CSS/JS), gehostet über GitHub Pages, verknüpft mit der Domain **angela-wolber.de** (IONOS).

## Seiten
- `index.html` – Startseite
- `ueber-mich.html` – Über mich
- `leistungen.html` – Leistungen, Hörproben & Preise
- `hoerbuecher.html` – Alle Hörbücher
- `kontakt.html` – Kontaktformular (Formspree)
- `impressum-datenschutz.html` – Impressum & Datenschutz

## Hörproben
MP3-Dateien gehören in den Ordner `hoerproben/` mit diesen Namen:
`hoerbuch.mp3`, `synchron.mp3`, `doku.mp3`, `gaming.mp3`, `trickstimme.mp3`, `werbung.mp3`, `werbung-gefuehlvoll.mp3`, `telefonansage.mp3`
